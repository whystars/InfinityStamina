# InfinityStamina
![Exiled](https://img.shields.io/badge/Exiled-latest-blue)

SCP: Secret Laboratory 服务器的无限体力插件，基于 Exiled 框架开发。

## 功能
- 永久禁用玩家体力消耗
- 自动恢复体力至最大值
- 可配置恢复间隔

## 安装
1. 将 `InfinityStamina.dll` 放入 `Exiled/Plugins` 目录
2. 配置 `config.yml`：
